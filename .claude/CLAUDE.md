# スポーツタイマー

## 概要

簡易的なタイマー。 `スタートボタン` `ストップボタン` は1つのボタンで押下するたびに切り替え。
`00:00:00` ~ `99:59:59` までをデジタル表示。
ストップした際の時間表示は次のスタートが押下されるまで固定。
記録機能はなし。

## 構成

* ブラウザアプリ
* HTML + JS + Tailwindcss
* localstrage 等の永続処理は利用しない
