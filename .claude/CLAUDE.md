# スポーツタイマー

## 概要

簡易的なタイマー。専用のボタンは持たず、画面(ウィンドウ)のどこをクリック/タップしてもスタート・ストップが切り替わる。
`00:00:00` ~ `99:59:59` までをデジタル表示。
ストップした際の時間表示は次にスタートするまで固定。
ストップするたびに計測結果を記録し、直近10件を履歴として一覧表示する(順位表示はなし)。

## 構成

* ブラウザアプリ
* HTML + JS + Tailwindcss
* 計測履歴(直近10件)の保存には localStorage を利用する。それ以外の永続処理は利用しない
